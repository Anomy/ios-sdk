public class NetworkUtils

Networking utilities used for performing REST operations into Watson services and parsing the input

  • requestAuthToken(_:serviceURL:apiKey:completionHandler:)

    Request a Watson Authentication token. Tokens expire after one hour.

    Declaration

    Swift

    public static func requestAuthToken(tokenURL: String, serviceURL: String, apiKey: String? = nil, completionHandler: (token: String?, error: NSError?) -> ())

    Parameters

    tokenURL

    The URL of the token authentication endpoint (e.g. https://stream.watsonplatform.net/authorization/api/v1/token)
    serviceURL

    The URL of the service for which you want to obtain a token (e.g. https://stream.watsonplatform.net/text-to-speech/api)
    apiKey

    The authentication string used for HTTP basic authorization.
    completionHandler

    The closure called when the token request is complete.
